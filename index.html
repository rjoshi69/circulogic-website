STAGE 01
Stage Title
Description text here.
Intelligence Engine

Five Scores. One Intelligence Engine.

Every score is maintained continuously, updated on every order and return event, and available at zero marginal cost at query time.

CRRS
Customer Return Risk Score

Scores every customer on return likelihood and behavioral risk. 0 to 100. Updated on every order and return event.

C-Score
Customer Value Score

True net customer value after return costs. Separates profitable customers from high-revenue margin destroyers.

E[L]
Expected Loss

The dollar cost of this customer's next return — computed before it happens. Drives every policy decision in the engine.

RAF-C
Return Adjustment Factor (Customer)

Customer-level risk multiplier. Catches the occasional-but-expensive returner that the base rate systematically misses.

RAF-P
Return Adjustment Factor (Product)

Product-level return risk independent of who is buying. Some products are inherently high risk regardless of customer history.

How It Works

From Signal to Policy in Milliseconds

Step 01

Evaluate customer and product risk across five independent dimensions. CRRS, C-Score, E[L], RAF-C, and RAF-P are each maintained continuously.
